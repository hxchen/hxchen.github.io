javascript 中contains方法的定义

2013-10-21
Web前端 JavaScript
约 69 字 预计阅读 1 分钟

文章目录

/**
 * 对大小写敏感的Contains方法
 * @param substring
 * @returns
 */
String.prototype.contains = function(substring)
{
    return this.indexOf(substring) != -1 ? true:false;
};

使用方法：a.contains(b)—-a中是否包含b
