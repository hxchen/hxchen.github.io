文章目录

一、Wordpress配置

下载

1

wget http://wordpress.org/latest.zip

解压

1

unzip latest.zip

进入目录

1

cd wordpress/

重命名

1

mv  wp-config-sample.php wp-config.php

更改配置

1

vi wp-config.php

请自行根据数据库设置进行配置文件设置。

二、PHP服务器配置

安装

1

yum install php php-mysql php-fpm

启动

1

service php-fpm start
