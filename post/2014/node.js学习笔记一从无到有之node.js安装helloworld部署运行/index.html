Node.js学习笔记（一）从无到有之Node.js安装、helloworld部署运行

2014-01-20
服务器 NodeJS
约 265 字 预计阅读 1 分钟

文章目录

一、下载

进入 官网，下载适合版本。

在此我选用的是64位windows：node-v0.10.24-x64.msi。

二、安装

① 安装下载下来的安装程序：node-v0.10.24-x64.msi，默认安装在 C:\Program Files\nodejs

安装完毕后，命令控制台运行 node -v,查看安装结果，显示版本号

②安装express。建议全局安装：

1

npm  install -g express

若是仅安装到当前目录则可使用

1

npm  install express

三、创建helloworld项目

express创建helloworld项目

1

express helloworld

进入项目目录，安装nodejs依赖模块.

1

cd helloworld && npm install

四、运行

运行helloworld项目

1

node app.js

浏览器访问：http://127.0.0.1:3000/ 即可看到显示结果
