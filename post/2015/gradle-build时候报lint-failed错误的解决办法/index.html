Gradle build时候报lint failed错误的解决办法

2015-04-27
Gradle
约 84 字 预计阅读 1 分钟

文章目录

这是因为程序在buid的时候，会执行lint检查，有任何的错误或者警告提示，都会终止构建，我们可以将其关掉。

修改build.gradle配置文件，在android里面添加

1
2
3
4
5
6

android{
    lintOptions {
            abortOnError false
    }
    ...
}
